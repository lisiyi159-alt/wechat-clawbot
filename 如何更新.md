# 日常更新与使用速查

## 一、同步最新代码（每次我改完功能后）

在微信开发者工具下方「**终端**」里，**一行一行**粘贴（PowerShell 不认 `&&`，所以分两行）：

```
git fetch origin claude/travel-photo-journal-app-q5c1uy
git reset --hard FETCH_HEAD
```

看到 `HEAD is now at ...` 即同步成功。

- 若本次改动**涉及后端云函数**（我会告诉你），再右键 `cloudfunctions/trips`（或对应云函数）→「上传并部署：云端安装依赖」。
- 开发者工具会自动编译；需要手动编译时按 `Ctrl+B`，或点右上角「普通编译」旁的圆形刷新箭头。

## 二、在自己手机上用（开发版预览）

1. 工具上方点「**预览**」（眼睛图标）→ 生成二维码
2. 手机微信（与登录工具同一个微信）→「+」→「扫一扫」→ 扫码打开
3. 之后在微信聊天列表下拉，「最近使用的小程序」里可再次打开

## 三、给家人用（体验版）

1. https://mp.weixin.qq.com →「管理」→「成员管理」→「体验成员」→ 添加家人微信
2. 工具上方点「**上传**」→ 填版本号（如 1.0.0）→ 上传
3. https://mp.weixin.qq.com →「管理」→「版本管理」→ 找到该版本 →「选为体验版」
4. 把体验版二维码发给已加入体验成员的家人，微信扫码即用

## 四、备份

- 数据存在云开发，不会按天删除；正式上线后第 15 天免费环境到期，需转付费续费（数据不丢）。
- 程序内首页「一键导出全部」可把照片存相册、文字进剪贴板，建议定期导出留存（程序每三个月也会提醒一次）。
