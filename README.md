# 家庭旅行手账 · 微信小程序

一个私密的家庭旅行相册：全家一起记录各地旅行的照片与文字，方便日后翻阅，分享给指定家人。

- **微信白名单访问**：只有被加入小程序「体验成员」名单的家人才能打开，其他人根本进不来，无需邀请码
- **全家共同记录 + 身份标识**：每位家人设置自己的称呼（妈妈、姥姥…）。一段旅行可由多人接力——首位显示「妈妈：XXXX」，后续显示「姥姥 补充：xxxx」，每条记录带各自的文字和照片
- **旅行足迹地图**：发布时「地图选点」，去过的地方会标在地图上，点标记即可回看
- **一键分享给家人**：转发卡片带封面图和标题，在家庭群里点一下就能发
- **一键导出留存**：把某段旅行所有人的照片保存到手机相册、文字复制到剪贴板，长期备份
- **定期提醒**：每三个月弹窗提醒你导出备份
- **零服务器**：基于微信云开发（数据库 + 云存储 + 云函数）

## 接入与运行步骤

1. **导入项目**：用「微信开发者工具」导入本目录，AppID 填你自己的小程序 AppID
   （把 `project.config.json` 里的 `REPLACE_WITH_YOUR_APPID` 换成你的，或在导入时填写）。

2. **开通云开发**：开发者工具顶部点「云开发」，新建一个环境，复制环境 ID，
   填入 `miniprogram/config.js` 的 `ENV_ID`。

3. **创建数据库集合**：在云开发控制台「数据库」中新建两个集合：
   - `members`（成员：角色 + 称呼）
   - `trips`（旅行记录，含多人 entries）

   权限可设为「仅创建者可读写」（所有读写都走云函数，已在服务端做权限校验）。

4. **上传云函数**：分别右键 `cloudfunctions/login`、`trips`，选择
   「上传并部署：云端安装依赖」。

5. **配置微信白名单（关键）**：登录 [微信公众平台](https://mp.weixin.qq.com)
   → 「管理」→「成员管理」，把要给看的家人微信加为**体验成员**。
   小程序保持「体验版」状态、不发布到线上，则只有名单内的家人能打开。

6. **首次进入**：第一次写记录时，会弹窗让你设置称呼（如 妈妈、姥姥）。
   你（第一个打开的人）是管理员，可删除任何人的内容；其他家人可新建旅行、
   补充记录、编辑/删除自己的记录。

## 地图选点说明

发布旅行时点「📍地图选点」会调起微信地图，选定后记录经纬度并显示在「足迹」页。

- 该能力依赖 `wx.chooseLocation`，已在 `app.json` 声明 `requiredPrivateInfos`。
- 部分小程序类目首次使用需在公众平台申请「地理位置」接口权限；
  若暂不可用，地点仍可手动填写文字（只是不会出现在地图上）。

## 权限模型

| 角色 | 谁 | 能力 |
| --- | --- | --- |
| admin | 第一个打开小程序的人（你） | 新建 / 编辑删除任何人的内容 / 查看 / 导出 / 分享 |
| member | 微信白名单内的其他家人 | 新建旅行 / 补充记录 / 编辑删除自己的记录 / 查看 / 导出 / 分享 |

> 访问权限由微信「体验成员」名单控制：没被加入名单的人打不开小程序，
> 因此程序内不再需要邀请码。

## 关于数据保存

云开发数据库与云存储中的数据**不会按天自动删除**，可长期保存。注意：

- 免费体验环境主要面向开发阶段，**小程序正式上线后第 15 天到期**，需转为付费续费
  （转付费时环境 ID 和数据都不变，不会丢失）。体验版长期使用同样建议尽早开通付费。
- 无论用什么平台，都建议用本程序的「导出留存」**定期把照片和文字备份到本地**。
  程序会每三个月提醒你一次。
